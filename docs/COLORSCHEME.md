# Quant BSPWM Color Scheme

## Philosophy

Quant is a dark-pastel colorscheme engineered for extended focus sessions.
The palette balances low-fatigue dark backgrounds with high-contrast pastel
accents. Every hue is hand-tuned for WCAG AA readability on the base background.

## Base Palette

| Role        | Name      | Hex       | Usage                          |
|-------------|-----------|-----------|--------------------------------|
| bg_dark     | void      | `#0e0c15` | Panel backgrounds, popups      |
| bg          | abyss     | `#16141f` | Primary background             |
| bg_light    | shadow    | `#1e1c2a` | Active window, selection bg    |
| bg_lighter  | mist      | `#2a2837` | Hover states, subtle borders   |
| fg_dim      | whisper   | `#6e6a86` | Comments, disabled text        |
| fg_muted    | haze      | `#908caa` | Secondary text                 |
| fg          | snow      | `#e0def4` | Primary text                   |
| fg_bright   | ice       | `#f2f0ff` | Highlighted text               |
